# Discoteca El Toro - Frontend

Página web del proyecto para la Discoteca El Toro.  
Hecho como trabajo de clase usando HTML y CSS.

## Páginas
- *index.html*: Página principal del sitio.
- *usuarios.html*: Formulario de registro de usuarios.

## Estilos
- *css/styleusu.css*: Estilos específicos para el registro de usuarios.

## Cómo usarlo
1. Descarga o clona el repositorio.
2. Abre index.html en tu navegador.
